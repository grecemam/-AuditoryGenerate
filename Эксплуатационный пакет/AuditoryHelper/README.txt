Приложение: AuditorHelper

Описание:
Десктоп-приложение для автоматизированного формирования аудиторного фонда колледжа на основе расписания и замен.

Функциональность:
- Загрузка и анализ расписания учебных групп.
- Учёт изменений в расписании.
- Распределение преподавателей по свободным аудиториям с учётом пожеланий по этажам.
- Учёт учебных практик, мероприятий и закреплённых кабинетов.
- Генерация Excel-файлов по корпусам (Нахимовский и Нежинская).
- Отправка готовых файлов на сервер.
- Поддержка фильтрации преподавателей и сохранения предпочтений.
- Интеграция с API и Telegram-ботом.
